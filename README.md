# CityMarks

CityMarks is a lightweight World of Warcraft retail addon for interface version
`120007` (WoW 12.0.7). It adds localized service labels and icons to supported
city maps.

## Features

- Labels, icons, or combined display modes
- Nearby-label grouping
- Primary-profession filtering
- Global label and icon sizing
- Per-city enable and scale controls
- Native AddOns settings panel
- English, Simplified Chinese, and Traditional Chinese
- Account-wide settings

Type `/citymarks` or `/cm` to open the settings panel. Commands do not mutate
settings.

## Installation

Copy the `CityMarks` directory into:

`World of Warcraft/_retail_/Interface/AddOns/`

## Supported maps

- Silvermoon
- Dornogal
- Tazavesh
- Valdrakken
- Dalaran
- Orgrimmar
- Stormwind

Map coordinates should be verified in game after major world-map revisions.
